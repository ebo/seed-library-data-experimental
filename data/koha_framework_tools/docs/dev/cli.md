# CLI

This project uses [Typer](https://typer.tiangolo.com/) and [Click](https://click.palletsprojects.com/) for CLI functionality. When the project is installed the cli is available at `koha_framework_tools`.

The full help contents can be visited with the help flag.

```bash
koha_framework_tools --help
```

The CLI itself is defined at `koha_framework_tools.cli`. New commands can be added there.
